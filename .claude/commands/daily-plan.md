# 오늘의 작업 계획

오늘 진행할 작업을 정리하고 계획을 세웁니다.

## 수행 작업

### 1. 현재 상태 확인

```bash
# 열린 이슈 확인
gh issue list --state open

# 진행 중인 PR 확인
gh pr list --state open

# 현재 브랜치 확인
git branch --show-current
```

### 2. 진행 중인 작업 파악

이슈의 체크박스를 확인하여 진행 상황을 파악합니다.

### 3. 오늘의 계획 제안

사용자에게 오늘 할 작업을 제안합니다:

```
## 📅 오늘의 작업 계획

### 진행 중인 작업
- Issue #3: 회원가입 API 구현 (3/5 완료)
  - [ ] UserService 작성
  - [ ] UserController 작성

### 오늘 목표
1. UserService 작성 (예상 1시간)
2. UserController 작성 (예상 1시간)
3. 테스트 및 PR 생성 (예상 30분)

### 예상 소요 시간
약 2시간 30분

### 시작하기
`/create-issue` 또는 기존 이슈 계속 진행
```

## 추가 기능

### 마일스톤 확인

CLAUDE.md의 마일스톤과 비교하여 전체 진행 상황을 알려줍니다.

```
## 📊 전체 진행 상황

### 1단계: 프로젝트 세팅 ✅ 완료
### 2단계: 사용자 관리 🔄 진행중 (40%)
### 3단계: 스터디 그룹 ⏳ 대기
### 4단계: 일정 관리 ⏳ 대기
### 5단계: 알림 시스템 ⏳ 대기
```
